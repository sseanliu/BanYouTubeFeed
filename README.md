# Ban YouTube Feed

A minimal Chrome extension that removes YouTube's recommendation feeds to help you stay focused.

## What it does

- **Homepage**: Hides the entire recommendation feed, filter chips, and Shorts shelf. Only the search bar and navigation remain.
- **Video pages**: Hides the right sidebar recommendations and endscreen suggestion overlays. The video player expands to full width.

## What's preserved

- Search bar and header
- Left sidebar navigation (subscriptions, library, history)
- Video player, title, description, and comments
- Channel pages, subscription feed, search results, playlists

## Installation

1. Clone or download this repository
2. Open Chrome and go to `chrome://extensions/`
3. Enable **Developer mode** (toggle in top-right)
4. Click **Load unpacked**
5. Select the `BanYouTubeFeed` folder
6. Go to youtube.com to verify it works
